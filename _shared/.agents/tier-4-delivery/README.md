# Tier 4 — Delivery Agents

**Parent**: [`@../README.md`](../README.md) | **Roster**: [`@../TEAM-CONFIG.md`](../TEAM-CONFIG.md) §I

**Mục đích**: 5 delivery-focused agents — bridge engineering output thành customer-ready deliverable. Phase 1 build skill card cho 2 critical (R-PM, R-SA).

## Roster

| ID | Role | Phase 1 Card? |
|---|---|---|
| R-PM | Project Manager | ✅ Critical path |
| R-SA | Solution Architect | ✅ Critical path |
| R-BA | Business Analyst | T6+ promote khi B2B Audit volume |
| R-UX | UX Designer | T9+ khi project có UI deliverable nhiều |
| R-TC | Technical Writer | T6+ khi case study + docs volume |

R-σ (Ms. Scribe) trong Tier 1 đảm nhận technical writing layer-3 — overlap với R-TC tạm thời. Phase 2 tách rõ.

## Distinction

- **R-PM** = "How do we deliver on time?" — planning, risk, stakeholder
- **R-SA** = "How do components fit together at system level?" — integration, scalability patterns
- **R-BA** = "What does the business actually need?" — requirements, user stories
- **R-UX** = "How will user feel/think?" — research, wireframes, design system
- **R-TC** = "Can someone other than us understand this?" — docs, API ref, guides
