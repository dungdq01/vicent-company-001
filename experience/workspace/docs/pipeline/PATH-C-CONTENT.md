# Path C — Content Piece (Content-led)

> Single content unit (YouTube, Substack, LinkedIn, TikTok, X thread, Audit report). Closes lỗ hổng B5.

**Trigger**: Content calendar (`16-brand-content-kit.md §5`) OR opportunity (trending topic).
**Owner**: Voice owner per voice (CEO Voice A, P3 Voice B, freelance Voice C).
**Authority**: Voice owner final.
**Priority**: 6 (lowest, per R-LCY-06) — flex schedule.

---

## Phase C0 — Topic Decision (15 min)

| Step | Action |
|---|---|
| C0.1 | Source: monthly theme OR opportunity OR repurpose request |
| C0.2 | Voice match? Technical → CEO; Business B2B → P3; MMO → freelance |
| C0.3 | Format? YT long / Substack / LinkedIn / TikTok / Audit / Thread |
| C0.4 | Channel target + repurpose plan (1 → 6 channels per `16 §6`) |

**Gate**: voice owner confirms availability for C2 production slot.

---

## Phase C1 — Knowledge Source (15 min)

3 source types:
- MAESTRO baseline / industry — read `knowledge/data/`
- Project case study (anonymized retro) — read `knowledge/docs/memory/`
- Original research (CEO conducts directly)

**Output**: source brief in `studio/content/{slug}/00-brief.md`.

---

## Phase C2 — Engine Draft (5–30 min)

**Optional engine assist** (for outline + first draft):
- R-CONTENT agent draft based on brief + voice descriptor
- Cost: <$0.10 per piece

**Skip engine** if CEO/owner prefers full manual (long-form essays, original analysis).

**Output**: `studio/content/{slug}/01-draft.md`.

---

## Phase C3 — Voice Owner Edit + Polish (1–6h)

Voice owner:
- Restructure for hook + flow
- Fact-check (R-MAS-06 anti-hallucination)
- Strip generic AI flavor (banned word list per `40-docs-rules.md`)
- Inject personal experience / opinion (this is what makes it Voice A/B/C distinctive)
- Final polish

**Output**: `02-final.md`.

---

## Phase C4 — DoD Check (15 min)

Per `_shared/standards/dod-per-deliverable.md` (or `business-strategy/17 §4`):

| Format | DoD essentials |
|---|---|
| YouTube | Hook 0-60s, ≤15 min, bilingual subs, pinned comment, thumbnail |
| Substack | ≤2000 words, hook lead, headers structured, CTA |
| LinkedIn | ≤2000 chars, hook line 1, structured, no banned phrases |
| TikTok | Hook ≤3s, vertical, captions, hashtags 3-5 |
| Thread | 5-12 tweets, hook tweet, evidence per tweet, CTA |
| Audit report | Branded, evidence-led, 1 actionable per insight, contact CTA |

**Gate**: all DoD pass. Else iterate or kill.

---

## Phase C5 — Publish + Repurpose (1h)

Per channel publish + cross-post per `16 §6`. 1 YouTube → ~12-15 content units across 6 channels.

**Output**: live URLs in `03-published.md`.

---

## Phase C6 — Measure (ongoing)

Per `business-strategy/03 §KPIs` content metrics:
- Views / impressions
- Engagement (comments, shares, CTR)
- Lead capture (email, follow)
- Sales attribution (when content drives Path B product)

Weekly: P3 dashboards. Monthly: CEO content retrospective.

---

## Phase C7 — Memory + High-Performer Save

Hook formulas from viral pieces → save to `studio/wisdom/high-performer-hooks.md`. Bottom 20% performers → analyze why → adjust strategy.
