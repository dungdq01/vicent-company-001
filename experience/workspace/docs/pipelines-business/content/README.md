## Cadence (Phase 1)

```
Mon  C0 review weekly batch
Tue  C1 draft article (newsletter)
Wed  C2 founder review + edit
Thu  C3 publish + distribute
Fri  C4 repurpose plan + social drops next week
```

Daily: R-CONTENT drops 1 social post (LinkedIn/Twitter VN) per R-MKT calendar.
