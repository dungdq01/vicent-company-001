# Expansion Playbook (E0-E3)

> **Offensive flow** — when client healthy + signal, propose upsell / cross-sell / longer retainer.
> **Distinct from CS3** (defensive — churn save when 🔴 health). E0-E3 = offensive when 🟢.

**Trigger**: Day-60 post-delivery (P10.1) OR retainer monthly health-check OR ad-hoc client signal.
**Owner**: R-CS (E0-E1) → R-AM (E2-E3).
**Authority**: COO for ≤ existing scope; CEO for ≥ 2× existing or strategic pivot.

---

## Phase Index

| # | Phase | Owner | Trigger | Output |
|---|---|---|---|---|
| E0 | Health Check | R-CS | Day-60 post-delivery OR monthly retainer | Health classification 🟢🟡🔴 |
| E1 | Opportunity Scan | R-CS + R-AM | E0 = 🟢 with signals | Top-3 ranked opportunities |
| E2 | Pitch | R-AM | Opportunity #1 selected | Custom proposal |
| E3 | Close-or-Renew | R-AM + R-LEG | Pitch sent + client decision | Outcome: expanded SOW / renewed only / lost |

---

## Distinct from CS3

| Aspect | CS3 (defensive) | E0-E3 (offensive) |
|---|---|---|
| Trigger | 🔴 churn risk | 🟢 + growth signal |
| Goal | Save | Grow |
| Tone | Empathetic, problem-solving | Confident, opportunity-led |
| Authority | COO + R-CS | R-AM + COO/CEO |
| Outcome | Stay/leave | Expand/renew/lost-but-cooled |

→ Same client may pass through both flows different times. Both legitimate.
