# Marketing pipeline (M0-M5)

> **Lead generation funnel for client acquisition**. Distinct from Path C (own content for thought leadership / brand).
> M0 strategic positioning → M5 optimize. Loops quarterly.

**Owner**: R-MKT + CEO (M0 + M5 strategic), R-MKT executes M1-M4.
**Cadence**: M0 quarterly, M1-M2 per campaign, M3-M4 per campaign, M5 monthly.

---

## Phase Index

| # | Phase | Owner | Trigger | Output |
|---|---|---|---|---|
| M0 | Positioning | CEO + R-BIZ | New quarter OR ICP shift signal | Stance + differentiation + anti-positioning |
| M1 | Channel Pick | R-MKT + CEO | After M0 OR new channel signal | Channel mix + budget split + frequency |
| M2 | Asset | R-MKT (brief) → R-CONTENT | M1 channel decided | Brief docs handed to Path C C0-C4 |
| M3 | Campaign | R-MKT | Asset published, campaign launches | Multi-channel execution log + UTM tracking |
| M4 | Measure | R-MKT + R-DA | Weekly + per-campaign close | Performance reports (CTR, CAC, MQL conversion) |
| M5 | Optimize | R-MKT + CEO | After M4 reveal trend | Updated calendar + ADR if structural shift |

---

## Distinct from Path C

| Aspect | Marketing M0-M5 | Path C content |
|---|---|---|
| Goal | **Lead generation** for client acquisition | Thought leadership, brand authority, own audience |
| Funnel position | Top of funnel (awareness → MQL → SQL handoff to Sales S0) | Top of funnel + retention + brand |
| Owner | R-MKT (orchestrates) + R-CONTENT (executes assets) | R-CONTENT primary, R-MKT calendar |
| Output | Lead pipeline, attribution, campaign reports | Articles, videos, podcasts, talks |
| Success metric | MQL count + CAC + reply rate | Engagement, audience growth, brand search |
| Pipeline trigger | Quarterly + per-campaign | Calendar-driven + opportunistic |

→ M2 hands off to Path C C0 for actual asset creation. Marketing doesn't write content — orchestrates.
