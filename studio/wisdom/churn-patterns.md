# Churn Patterns

> Per P10.2 + P10.4. Lost clients (whether at Day 90 or annual non-renewal) → exit interview → root cause → pattern saved here.
>
> When pattern recurs across 3+ clients → adjust ICP / pricing / delivery (likely ADR + memo to `business-strategy/`).

---

## Format

```
## YYYY-MM-DD · client-id (anonymized) · trigger
**Sprint tier**: A / B / C / D
**ICP**: A / B / C / D / E
**Industry**: I0X
**Lost at**: P10.2 (Day 90) | P10.4 (annual) | mid-Sprint cancellation
**Root cause**: 1-2 sentences — transferable, NOT project-specific
**Pattern matches**: existing pattern# OR new
**Action**: adjustment proposed (ICP / pricing / delivery / agent / scope)
```
